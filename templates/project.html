Project/Team Name: AskBudget

Code: https://github.com/budgethack/askbudget

Project Description Data Story

AskBudget aims to improve the delivery of information contained within the State Budget Papers. We wanted to create a platform to enable users to explore the language of the budget with a focus on issues that are important to them.

    To achievie this aim we employed the following techniques:
  • Converted the budget papers into a machine readable format.
  • Performed entity detection to learn about the key concepts in the various sections and identify the keywords.through the use of our own algorithims and IBM Watsin text-analysis API.
  • Data is then fed into elastic search.
  • Wrote queries to sort data and return relevant results.
